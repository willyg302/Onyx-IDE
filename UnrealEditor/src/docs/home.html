Onyx UnrealScript Editor [ Version 1.0 ]

What is Onyx?

Onyx is a cross-platform UnrealScript editor written in Java. It is designed with ease-of-use and portability in mind, requiring very little memory and capable of running on any machine with a JVM installed.

Why Choose Onyx?

Whether you work on the PC or the Mac, the Onyx experience won't change. That's because Onyx was build around the idea of cross-platform support: there's no installer or Mac disk image, just a Java distributable. You place the folder where you want it, maybe create a shortcut to the JAR that you can stick in the Start Menu or Applications stack, and that's it.

Onyx is far from becoming a robust "industry standard" editor, but feedback from the gaming community is sure to change that quickly. And not only do I welcome feedback, if there are any questions or problems I am willing to help solve them.

Oh, and did I mention...Onyx is free?

Requirements

- Java 1.4 or greater (works best with Java 1.6)
- 10MB of available hard-disk space for system files
- 512MB of RAM (at least 1GB recommended)

Features

- Works on Windows AND Mac (Linux untested)
- 16 beautiful themes, made possible with JTattoo
- Class tree viewer links to UnCodeX online
- UnrealScript syntax highlighting
- Support for multiple open files in separate tabs
- In-editor compiling, options to run UDK Game (Windows only)
- Tab/space conversion and cleanup
- Toggle seeing whitespace (tabs/spaces)
- Bracket auto-complete and folding
- Drag-and-drop to open files, open multiple files, open from the command line
