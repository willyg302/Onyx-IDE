Edit Menu

OptionShortcutDescription
UndoZ Undoes the most recent edit on the current tab.
RedoY Redoes an edit (if any) on the current tab.
CutX Moves selected text to the clipboard.
CopyC Copies selected text to the clipboard.
PasteV Inserts clipboard contents at the current cursor location.
Find...F Brings up the Find dialogue. For more information on what this can do, see "Find and Replace" in the Advanced Features section.
Find NextF3 If there is something currently in the Find dialogue, this will find its next instance. Otherwise it will open the Find dialogue.
Replace...R Brings up the Replace dialogue. For more information on what this can do, see "Find and Replace" in the Advanced Features section.
Go To...G Brings up the Goto Line dialogue. Enter a line number and Onyx will jump to that line number for you.
Tabs See the Tabs Submenu page.
Line termination See the Line Termination Submenu page.
Complete CodeCtrl + Space Calls the autocomplete system on the word or phrase you are currently typing. For more information on what this can do, visit "Autocomplete" in the Advanced Features section.
Select AllA Highlights all contents of the current tab.
